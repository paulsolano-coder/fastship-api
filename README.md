# Fastship API

API desarrollada con Node.js, Express y MongoDB.

## Tecnologías
- Node.js
- Express
- MongoDB

## 📁 Estructura del proyecto

fastship-api/
│
├── src/
│   ├── controllers/
│   ├── models/
│   ├── routes/
│   └── app.js
│
├── config/
├── node_modules/
├── package.json
└── .gitignore

## Instalación
npm install
npm run dev
